tweet cleaner
==
time based tweet cleaner.

setup
--
````
pyenv install 3.4.1
pyenv virtualenv 3.4.1 tweet-cleaner-3.4.1
pip install -r requirements.txt
````
````
cp clean.configuration.sample clean.configuration
vi clean.configuration
````

kick
--
````
python ./clean.py
````
